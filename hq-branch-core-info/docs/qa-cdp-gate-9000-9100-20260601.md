## `cargo check` Only If `apps/api` Changed

```powershell
Set-Location "D:\Lucas Core v0.1"
$ApiTouched = @(
  git diff --name-only -- apps/api
  git diff --name-only --cached -- apps/api
) | Where-Object { $_ }

if ($ApiTouched.Count -gt 0) {
  cargo check --manifest-path apps/api/Cargo.toml *>&1 | Tee-Object -FilePath "$Evidence\05-api-check.txt"
} else {
  "SKIPPED: apps/api not touched" | Out-File -Encoding utf8 "$Evidence\05-api-check.txt"
}
```

## Browser Cleanup Confirmation

The prepared helpers close the browser themselves. Verify that no evidence-scoped browser process remains:

```powershell
$EscapedEvidence = [Regex]::Escape($Evidence)
Get-CimInstance Win32_Process |
  Where-Object {
    ($_.Name -match 'chrome|msedge') -and
    $_.CommandLine -match $EscapedEvidence
  } |
  Select-Object ProcessId,Name,CommandLine |
  Format-List | Out-String | Out-File -Encoding utf8 "$Evidence\09-browser-cleanup.txt"
```

Pass criteria:

- `09-browser-cleanup.txt` is empty or contains no live browser process tied to `$Evidence`
